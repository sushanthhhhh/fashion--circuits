SOS Signal Flasher
Uses an LED to blink the Morse code for SOS (… --- …).

📌 Wiring

  • LED anode → Arduino pin 9 (via 220Ω resistor)
  • LED cathode → GND
  • Arduino UNO R3
  • LED (any color)
  • 220Ω resistor
int led = 9;
void setup() { pinMode(led, OUTPUT); }
void loop() {
  for(int i=0;i<3;i++){ digitalWrite(led,HIGH); delay(300); digitalWrite(led,LOW); delay(300); } // S
  delay(700);
  for(int i=0;i<3;i++){ digitalWrite(led,HIGH); delay(900); digitalWrite(led,LOW); delay(300); } // O
  delay(700);
  for(int i=0;i<3;i++){ digitalWrite(led,HIGH); delay(300); digitalWrite(led,LOW); delay(300); } // S
  delay(5000);
}
Digital Hourglass
Lights a new LED every 10 minutes to track time.

📌 Wiring

  • LED 1 → pin 9 (via 220Ω)
  • LED 2 → pin 10 (via 220Ω)
  • LED 3 → pin 11 (via 220Ω)
  • LED 4 → pin 12 (via 220Ω)
  • Cathodes all to GND
  • Arduino UNO R3
  • 4 LEDs
  • 4× 220Ω resistors
int leds[] = {9,10,11,12};
void setup() { for(int i=0;i<4;i++) pinMode(leds[i],OUTPUT); }
void loop() {
  for(int i=0;i<4;i++){ digitalWrite(leds[i],HIGH); delay(600000); } // 10 min = 600,000 ms
  for(int i=0;i<4;i++) digitalWrite(leds[i],LOW);
  delay(1000);
}

Remote Fan Control
Uses a button to safely turn a DC fan on/off via relay.

📌 Wiring

  • Relay IN → pin 7
  • Push button → pin 2 (with INPUT_PULLUP)
  • Relay VCC/GND as above
  • Fan connected to COM and NO
  • Arduino UNO R3
  • Relay module
  • Push button
  • DC fan
int relay = 7, button = 2;
boolean state = false;
void setup() { pinMode(relay,OUTPUT); pinMode(button,INPUT_PULLUP); }
void loop() {
  if(digitalRead(button)==LOW){ delay(50); if(digitalRead(button)==LOW){
    state = !state; digitalWrite(relay,state); while(digitalRead(button)==LOW); delay(50);
  }}
}

High‑Heat Warning LED
Red LED turns on when temperature > 30°C.

📌 Wiring

  • DHT11 data → pin 2
  • DHT11 VCC → 5V, GND → GND
  • Red LED anode → pin 9 (via 220Ω)
  • LED cathode → GND
  • Arduino UNO R3
  • DHT11
  • Red LED
  • 220Ω resistor
#include <DHT.h>
DHT dht(2,DHT11);
int led = 9;
void setup() { pinMode(led,OUTPUT); dht.begin(); }
void loop() {
  float t = dht.readTemperature();
  if(t > 30) digitalWrite(led,HIGH); else digitalWrite(led,LOW);
  delay(2000);
}

Parking Sensor
Buzzer beeps faster as an object gets closer.

📌 Wiring

  • HC-SR04 VCC → 5V, GND → GND
  • Trig → pin 9, Echo → pin 10
  • Buzzer (+) → pin 8, (-) → GND
  • Arduino UNO R3
  • HC-SR04
  • Buzzer
int trig=9, echo=10, buzzer=8;
void setup() { pinMode(trig,OUTPUT); pinMode(echo,INPUT); pinMode(buzzer,OUTPUT); }
void loop() {
  digitalWrite(trig,LOW); delayMicroseconds(2); digitalWrite(trig,HIGH); delayMicroseconds(10); digitalWrite(trig,LOW);
  long duration = pulseIn(echo,HIGH);
  int distance = duration * 0.034 / 2;
  if(distance > 0 && distance < 50){
    int beepDelay = map(distance,0,50,50,500);
    tone(buzzer,1000,beepDelay); delay(beepDelay);
  }
  delay(50);
}

📌 Wiring

  • HC-SR04 VCC → 5V, GND → GND
  • Trig → pin 9, Echo → pin 10
  • Arduino UNO R3
  • HC-SR04
int trig=9, echo=10;
void setup() { pinMode(trig,OUTPUT); pinMode(echo,INPUT); Serial.begin(9600); }
void loop() {
  digitalWrite(trig,LOW); delayMicroseconds(2); digitalWrite(trig,HIGH); delayMicroseconds(10); digitalWrite(trig,LOW);

}

Visual Noise Meter
The louder the sound, the more LEDs light up.

📌 Wiring

  • Sound sensor OUT → A0 (analog)
  • 3 LEDs → pins 9,10,11 via 220Ω resistors
  • Arduino UNO R3
  • Sound Sensor
  • 3 LEDs (red, yellow, green)
  • 3× 220Ω resistors
int sound = A0;
int leds[] = {9,10,11};
void setup() { for(int i=0;i<3;i++) pinMode(leds[i],OUTPUT); }
void loop() {
  int val = analogRead(sound);
  int level = map(val,0,1023,0,3);
  for(int i=0;i<3;i++) digitalWrite(leds[i], i < level ? HIGH : LOW);
  delay(100);
}
